Event Hawk

A web search engine to find events in your area! Personalize your calendar, buy tickets to concerts, and more. Tech Stack: PHP, Javascript, HTML/CSS, MySQL. Features: full CRUD functionality, AJAX calls to the TicketMaster API.



StuOrgCentral

StuOrgCentral is a fully functioning CRUD application that allows USC students to filter and find RSO Events. Tech Stack: MERN.



CommonTune

CommonTune is a web app that acts as a portal for jazz musicians to find each other in the area. They will be able to friend each other, create bands (groups), and message each other. Once a band is formed, the web app will auto-generate a list of jazz standards that everyone in the group knows, as well as recommend tunes based on their overall repertoire. Tech Stack: MERN.
